# HyperFORM

`HyperFORM` is a `FORM` package for the analytic, parametric integration of
Feynman integrals using hyperlogarithms. It implements the algorithms of
Erik Panzer's `HyperInt`, ported to and optimized for `FORM`.

> **Note:** This copy of the repository is **not** kept up to date. For the
> latest version of the code, please refer to the original repository at
> <https://github.com/adamkardos/HyperFORM>.
